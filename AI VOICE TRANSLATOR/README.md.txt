# 🎙️ Voice AI Translator

This is a simple Voice AI Translator built with Python.

**Features:**
- 🎤 Speech Recognition (mic input)
- 🌐 Google Translate (deep_translator)
- 🔊 Text-to-Speech (gTTS)

**How to run:**
1. Install requirements:  
   ```
   pip install speechrecognition pyaudio deep-translator gtts
   ```

2. Run the script:  
   ```
   python translator.py
   ```

3. Speak your sentence → choose source & target languages
